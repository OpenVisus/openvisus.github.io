Install & configure

Shell

pip install --upgrade OpenVisus

Load a dataset

LoadDataset(url) returns a PyDataset. The URL can be a path to an .idx file or an HTTP URL to a ViSUS server.

Python

from OpenVisus import LoadDataset

db = LoadDataset("./data/my_dataset.idx")
# or: db = LoadDataset("http://example.com/visus/dataset.idx")

Inspect fields, bounds, and logic size

Use these before choosing slices or timesteps. getPointDim() is 2 or 3 for native 2D vs 3D datasets. Logic coordinates are integer indices; you can also use normalized fractions (see below).

Python

print(db.getPointDim())         # 2D or 3D Data (excluding Time)
print(db.getFields())           # channel / variable names
print(db.getLogicSize())        # size in logic indices
print(db.getMaxResolution())   # finest refinement level
print(db.getTimesteps())        # list of timesteps available(can be long)

read() & getLogicBox: floats vs ints

read() passes ranges to getLogicBox. Each bound uses its Python type: float → normalized fraction along that axis; int → logic index. Omit an axis to use the full domain along that axis. The 2D and 3D sections below apply the same rules—only which axes you constrain changes.

2D examples

Use when getPointDim() == 2, or when you read a planar slice through 3D data by fixing z (or x / y) to a thin slab [k, k+1]. Omit z only for native 2D IDX.

2D crop — normalized X, Y

data = db.read(x=[0.0, 0.9], y=[0.0, 0.9])

2D crop — logic indices on X, Y

data = db.read(x=[0, 100], y=[0, 100])

3D volume → one XY plane (fix Z)

data = db.read(x=[0.0, 0.5], y=[0.0, 0.5], z=[512, 513])
data = db.read(field="temperature", z=[100, 101])  # full XY at one Z

Logic box — 2D window

p1, p2 = db.getLogicBox(x=[0.0, 0.25], y=[0.0, 0.25])
data = db.read(logic_box=(p1, p2))

3D examples

Use when getPointDim() == 3 and you need a subvolume (not just a single slice). Constrain x, y, and z together.

3D subvolume — normalized box

data = db.read(
    x=[0.0, 0.1],
    y=[0.0, 0.1],
    z=[0.0, 0.1],
)

3D subvolume — logic indices

data = db.read(x=[0, 128], y=[0, 128], z=[400, 512])

Logic box — 3D corners from normalized ranges

p1, p2 = db.getLogicBox(x=[0.0, 0.25], y=[0.0, 0.25], z=[0.0, 0.25])
data = db.read(logic_box=(p1, p2))

Timestep, quality, and progressive refinement

These read() options combine with any window: native 2D, an XY slice through 3D (fixed z), or a 3D brick. If time is omitted, the dataset’s current/default timestep is used. quality is non-positive (0 = full resolution); negative values request coarser data. See the read docs. Some Linux builds support a size argument for an approximate memory budget.

Timestep

2D — slice at time=0 vs time=1

plane_t0 = db.read(z=[512, 513], time=0)
plane_t1 = db.read(z=[512, 513], time=1)

2D — native X/Y only

img = db.read(x=[0, 512], y=[0, 512], time=2)

3D subvolume

brick_t0 = db.read(x=[0, 64], y=[0, 64], z=[0, 64], time=0)
brick_t1 = db.read(x=[0, 64], y=[0, 64], z=[0, 64], time=1)

Quality

2D slice — coarser (faster) preview,entire X/Y

plane_coarse = db.read( quality=-3)

2D window — X/Y bound 

img_coarse = db.read(x=[0, 256], y=[0, 256], quality=-2)

3D brick — X/Y/Z bound

brick_coarse = db.read(x=[0, 64], y=[0, 64], z=[0, 64], quality=-2)

Progressive refinement

With num_refinements > 1, read() returns a generator that yields the same region at progressively finer resolutions; use num_refinements=1 (default) for a single numpy array.

2D — native dataset (x/y only)

for data in db.read(x=[0, 256], y=[0, 256], num_refinements=3):
    print(data.shape, data.dtype)

2D — XY slice through 3D (fixed Z)

for plane in db.read(x=[0, 256], y=[0, 256], z=[0, 1], num_refinements=3):
    print(plane.shape)

3D brick

for brick in db.read(x=[0, 128], y=[0, 128], z=[0, 32], num_refinements=3):
    print(brick.shape)

Optional: plot a 2D slice

After a read that returns one slab, squeeze and pass to Matplotlib.

Python

import matplotlib.pyplot as plt
import numpy as np

data = db.read(z=[512, 513], field=db.getFields()[0])
arr = np.asarray(data).squeeze()
plt.imshow(arr, cmap="viridis", aspect="auto")
plt.colorbar()
plt.show()
