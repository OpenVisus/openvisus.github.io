OpenVisus

Infrastructure

How OpenViSUS Works. The OpenViSUS infrastructure acts as an invisible acceleration layer between raw data and the analysis tools scientists already use.

The IDX Data Format

Our proprietary (but open-source) file layout avoids data replication and instills spatial and hierarchical locality. Data is returned in a coarse-to-fine order, enabling instant, progressive rendering regardless of network bandwidth. OpenVisus also allows the user to specify a fixed resolution to render the data.

Deployment Flexibility

OpenViSUS supports public cloud object storage (Amazon S3, Google Cloud, Azure). However, recognizing the prohibitive egress costs of the cloud, we also offer secure, containerized "Snow Globe" (On-Premise) deployments using Docker, Singularity, and Kubernetes for high-speed local processing inside secure facilities. OpenVisus can also be deployed on a local machine or a cluster.

Analysis Integrations

We provide a C++ core wrapped with a robust Python API (openvisuspy). This enables direct "IDX-to-NumPy" bridges and integrations into environments like Jupyter, providing seamless functionality for machine learning pipelines without altering existing workflows.
